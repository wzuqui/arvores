# Exemplos de árvores binárias

Os resultados devem ser no console

```typescript
// Desenhando a árvore abaixo:
//
//    '+'
//   /   \
//  'a'   '*'
//       /   \
//     'b'   '-'
//          /   \
//        '/'   'e'
//        /  \
//      'c'  'd'
//
// Resultado simétrico tranversal deve ser:
// (a + (b * ((c / d) - e)));

// Desenhando a árvore abaixo:
//
//      '!'
//    /    \
//  'O'    'o'
//        /    \
//      'l'    'd'
//           /     \
//        'm'      'n'
//        /  \     / 
//      'á'  ' ' 'u'
//
// Resultado pos ordem transversal deve ser:
// Olá mundo!
// Altura da árvore: 5
```
